In questa repository si trovano dei piccolissimi programmi in C++ che svolgono semplici operazioni matematiche:

* Programma: chiede di inserire un numero intero e, a seconda del numero inserito, esegue diverse operazioni.
* Programma 2: calcola la somma e la media aritmetica di 10 numeri, più un calcolo speciale!



###### Come avviare il programma:

Per avviare il programma, compilare il relativo file .cpp usando un compilatore C++ (come g++ o Visual Studio) e avviare l'eseguibile generato.
